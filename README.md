# Морской бой
Всемиизвестная и легендарная игра, в боях в которой сходятся флотилии военных кораблей, под началом лучших командиров - **морской бой**

## Getting Started
```
$ npm install
$ npm start
```

## Использованные технологии:
- HTML 5/ CSS
- JavaScript
- React
